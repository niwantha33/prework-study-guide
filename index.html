HTML

Hold the content of the webpage or performs a specific task

  • HTML Elements: provides important information about a website to the browser
  • "!DOCTYPE html" ( Not case sensitive)

    • DOCTYPE is the first line of the HTML code, and this declaraton is an instruction to the web browser about version of the HTML page and it will ensure that the web page is parsed correctly to all the web browsers.

    Example: Syntax for HTML 5 <!DOCTYPE HTML> There are three seperate rendering modes, available accross the browsers;

    • Full Standards Mode renders the page as per the W3C web standards
    • Quirks Mode Non-standards mode
    • Almost Standards Mode almost close to the full standard mode but support for a certain extend to the quirks
  • Encoding HTML File

    • Always need to specify the encoding by using meta element charset attribute or http-equiv and content recomended to delcare after the opening <head> tag

                
              <!DOCTYPE html>
              <html lang="en">
              <head>
              <meta charset="utf-8">
